# Art Portfolio website

A responsive personal art portfolio website created to showcase the artwork of Bitanya Tekle.
The website presents her art, idea, and contact information in a clean and minimal layout.

## Features

1.Responsive design for desktop, tablet, and mobile

2.Artist introduction section

3.Portfolio gallery displaying artworks

4.Contact section with email and Telegram

5.Instagram profile link

6.Hamburger navigation menu for mobile devices

## Built with

-HTML5

-CSS3

-JavaScript

-Media Queries (Responsive Design)
